# lostark_example

회사에서 지원해주는 ccl(Creative challenge league) 프로젝트의 시뮬레이터 코드
골드 잃고 초파고 만드는 팀
박종철(시뮬레이터), 이용철(학습), 박주형(데이터 및 영상), 한민기(gui)

로스트아크 공식 가이드, 유투브, 초파고를 참고해서 만들었습니다
